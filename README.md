# Crestline Studio

Portfolio project for a fictional design and branding agency built with Next.js 15, React 19, TypeScript, Tailwind CSS v4, Framer Motion, and MDX.

## Stack

- Next.js 15 App Router
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- React Hook Form + Zod
- MDX content with `gray-matter` and `next-mdx-remote`

## Getting Started

```bash
pnpm install
pnpm dev
```

## Scripts

```bash
pnpm dev
pnpm build
pnpm start
pnpm lint
pnpm typecheck
```
